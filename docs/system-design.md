# System Design - AI E-Learning Platform

## Architecture Overview
- Frontend: React.js (SPA)
- Backend: Node.js + Express.js (REST API)
- Database: MongoDB (NoSQL)
- Auth: JWT Tokens
- Realtime: Socket.IO
- State: Redux / Context API
- Deployment: Docker + Kubernetes
- Media: Cloudinary / Firebase

## System Flow
1. User registers/logs in (JWT issued)
2. AI engine captures user interests
3. Personalized dashboard shown
4. Student enrolls in courses
5. Progress tracked in real-time
6. Instructor creates and manages courses
7. Admin approves content and manages users
